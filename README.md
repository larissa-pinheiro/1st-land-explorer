<h1 align="center">
	Móveis customizados
</h1>

Primeiro projeto desenvolvido durante o curso Explorer da <a href="https://www.rocketseat.com.br">Rocketseat</a>.

Como contribuição própria, adicionei o conceito de variáveis CSS e responsividade simples.
